## 深度优先搜索算法(DFS)

Deapth First Search

1.设有n个整数的集合{1,2,3,...,n}，全为正整数且没有重复，请打印出所有长度为3的子集。

【输出】

{1,2,3}, {1,2,4},...,{1,2,n},

{1,3,4}, {1,3,5},...,{1,3,n},

{1,4,5}, {1,4,6},...,{1,4,n},

...,

{2,3,4}, {2,3,5},...,{2,3,n},

{2,4,5}, {2,4,6},...,{2,4,n},

{2,5,6}, {2,5,7},...,{2,5,n},



2.给一个数组以及一个数字X，这个数组里全为正整数且没有重复，在这个数组中可能存在几个数之和等于X，请打印出所有这些数的组合，用[]隔开。

- 比如数组{1, 4, 6, 2}, X = 6

    输出：[6], [4,2]
- 数组{1, 4, 6, 2}, X = 7

    输出：[1,6], [1,4,2]
